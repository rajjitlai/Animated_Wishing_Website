# Pen Writing Animation

An animated greeting card with a typewriter text effect and image flipbook slideshow.

## Features

- Typewriter animation with pen writing/erasing effect
- Image flipbook with smooth crossfade transitions
- Responsive design for desktop and mobile
- No external dependencies — pure HTML, CSS, and vanilla JavaScript

## Demo

Open `index.html` in any modern browser to view the greeting card.

## Project Structure

```
wish/
├── index.html          # Main HTML page
├── css/
│   └── style.css       # Styles and animations
├── js/
│   └── index.js        # Typewriter and flipbook logic
├── images/
│   ├── bg.png          # Background image
│   └── 1-5.png         # Flipbook photos
├── assets/
│   └── paper.png       # Paper texture
├── LICENSE             # MIT License
└── README.md           # This file
```

## How to Run

1. Clone or download this repository.
2. Open `index.html` in a web browser.

No build step or server required.

## Tech Stack

- HTML5
- CSS3 (animations, flexbox, media queries)
- Vanilla JavaScript (ES6 classes)

A beautiful, sentimental animated greeting card featuring a realistic typewriter effect with a trailing pencil animation and a smooth image flipbook. Perfect for Father's Day or any special occasion.

[Live Demo](https://rajjitlai.github.io/Animated_Wishing_Website/) *(Update this if you have a GitHub Pages deployment)*

## ✨ Features

- **Realistic Writing Animation**: A pencil icon that trails the text as it's typed and flips to its eraser side when deleting.
- **Image Flipbook**: A smooth, fading image carousel to showcase cherished memories.
- **Semantic HTML5**: Built with accessibility and SEO in mind.
- **Modern CSS**: Uses CSS Custom Properties (Variables) for easy theming and responsive design.
- **Responsive**: Fully optimized for Desktop, Tablet, and Mobile devices.

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser to run this project.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rajjitlai/Animated_Wishing_Website.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Animated_Wishing_Website
   ```

3. **Open the project:**
   Simply open `index.html` in your favorite browser.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties, keyframe animations, and Flexbox.
- **JavaScript (Vanilla)**: Custom `TypeWriter` logic and image rotation.
- **FontAwesome**: For the pencil and UI icons.
- **Google Fonts**: "Dancing Script" for the handwritten aesthetic.
